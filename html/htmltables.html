HTML Tables

Honestly HTML tables are a huge pain in the ass. I think it would we way faster to use a different program and export it as html, or simply use an image of the table and input it to the page.


Table Element Type Meaning
table Tag Parent of table data
th Tag Table Header
tr Tag Starts new table row
td Tag Table data
thead Tag Header for table
tbody Tag Separates body of table from head and foot
tfoot Tag Separates final part of table
