HTML Code

html code is made up of various elements. These elements are described by tags enclosed by pointed brackets. Within these brackets is the element name as well as any attributes associated with the element. Almost all styling for html is done outside of the main html document in a css stylesheet.

Code Type Meaning
!DOCTYPE html Self-Closing Tag Marks file as html
html Tag Says that following code is html
head Tag Undisplayed html code that is used by browser
title Tag Name displayed in top tab of browser
body Tag Used to mark all visible site code
div Tag Used to separated page into grouped elements
p Tag Block of plain text
span Tag Separate small pieces of content within a line to be an editable element
em Tag Italicizes
strong Tag Boldens
br Self-closing Tag Breaks a line of text
ul Tag Unordered (bulleted) list
ol Tag Ordered (numbered) list
li Tag Element of list
img Self-closing Tag Image
video Tag Video
a Tag Anchor. Makes a hyperlink
! -- Other Comment
style Tag Used to write CSS code within an HTML head
