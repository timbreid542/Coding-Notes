HTML Attributes

In HTML attributes are put inside of element tags in order to change the behavior of the element.


Code Meaning Value Associated elements
src Marks the source of an image or video URL, File Path img, video
id Name Specifically mark an element for individual css styling Any
alt Assigns a description on an element in case of loading failure Description img
href Links to a different page URL, File Path a, link
colspan Widens the cell by N column lengths Integer td,th
rowspan Lengthens the cell by N row lengths Integer td,th
type Describes type of file being linked text/css, other file types link
rel Describes the relationship between linked file and current file stylesheet link
class Assigns the element to a stylable class Class Name Any
role changes screen reader communication complementary/note/alert/article/etc Any
