Command Line and Git

Code Meaning
ls List folders and files in current directory
pwd Prints working directory (address of where you are)
cd Changes directory
.. Backs up one level in filesystem
mkdir Makes a new folder within working directory
touch Creates a new file in the working directory
init Initializes a program
git init Initializes the repository as a git project
git add Adds file to staging area
git status Shows the git status of all files in the current working directory
git diff changes Checks the difference between the working area and staging area
git commit -m '' Commits the staging area permanently
git log Shows the log of previous commits
echo Prints a string to the terminal
rm Removes a file from the working directory
