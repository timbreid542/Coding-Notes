CSS Transitions

Code Meaning Value
transition-property Names the transitioning property Any CSS animateable property/"all" sets parameters for all changing properties
transition-duration Sets the time for transition change #s/ms
transition-delay Waiting time before starting a transition #s/ms
transition-timing-function Sets pace of transition ease/ease-in/ease-out/linear/ease-in-out
transition Combination of property, duration, timing function, and delay rules property #s function #s. Can add consequent sets separated by commas
