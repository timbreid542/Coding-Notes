CSS Box Model

Position is mediated within html and css via the box-model. The model includes the contents size (width and height) and its padding, border, and margin.

Top and bottom margins collapse, side margins do not without being told
Content-box = default box. Defined by width, length, padding, border, and margin
Border-box = Box solely defined by length and width. All border and padding exists inside these defined dimensions

Code Meaning Value
width Width of the content area #px
width Width of the content area #px
min/max-width Sets limits on content area #px
height Height of content area #px
min/max-height Sets limits on content area #px
border specifies thickness, style, and color of border {thin, medium, thick} {10 renderable styles} {color}
border-top/right/bottom/left Sets border on only one side Same as border values
border-radius Makes corners of border curved #px, where # is the radius of the implied circle. 100% will make a circular container
padding Amount of space between the content and border #px(all) OR #px(top/bottom) OR #px(top), #px (left/right), #px(bottom) OR #px(left/right) OR #px(top), #px(right), #px(bottom), #px(left)
margin Size of element beyond border #px(all) OR #px(top/bottom) OR #px(top), #px (left/right), #px(bottom) OR #px(left/right) OR #px(top), #px(right), #px(bottom), #px(left)
margin-top/right/bottom/left Sets margin on only one side #px
overflow Sets behavior of element that extends beyond container hidden, scroll, visible
visibility Allows elements to be hidden from the viewer hidden/visible
box-sizing specifies the way a box is to be interpreted content-box, border-box
