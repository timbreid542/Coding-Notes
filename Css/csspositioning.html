CSS Positioning

Static Positioning: Default position. Left to right, up to down

Relative Positioning: Similar to static, but with up/down/left/right modifiers to move it a set amount from default position

Absolute Positioning: All other elements will ignore the positioning of this element, its set position is relative to its parent.

Fixed Positioning: The element is fixed to a certain part of the screen. It will move when the screen is scrolled. All other elements will ignore it.

Flex/Inline Flex Positioning: Used to set an element as a flexbox container

Code Meaning Value
position The default position of an element static/relative/absolute/fixed
top/bottom/left/right Adds an offset to the default position of an element #px
z-index Controls depth of element; what is in front of and behind Integer. Higher number = More forward
display Directs whether an element can share horizontal space inline/block/inline-block
float Moves the element as far right or left as possible on the page Left/Right
clear Makes a side avoid collision with floated objects left/right/both/none
justify-content Sets horizontal location of flex items within the container flex-start/flex-end/center/space-around/space-between
align-items Sets vertical location of flex items within the container flex-start/flex-end/center/baseline/stretch
align-content Sets vertical location of multiple rows within a flex container flex-start/flex-end/center/space-around/space-between
flex-grow/shrink Assigns ratios of growth to items within a flex container Integer
flex-basis The value at which flex-grow/shrink go into effect #px
flex shorthand for flex-grow/shrink/basis ratio, ratio, size
flex-wrap Sets behavior for when items outsize flex container wrap/wrap-reverse/nowrap
flex-direction Sets direction of flex items row/column/row-reverse/column-reverse
flex-flow Combination of flex-wrap and flex-direction direction, wrap
