CSS Typefaces

Code Meaning Value
font-family Typeface Any Font. Preferably websafe
@font-face Allows for a custom/internet typeface src: url(link to file)
text-transform Transforms the elements text Uppercase, Lowercase, Capitalize
font-size Applys a font size # of px
font-style Italicizes text Normal, Italic, Oblique
font-weight Relative boldness of font bold, bolder, or integer value
text-shadow Puts shadow effect under text n/* offset-x | offset-y | blur-radius | color */
word-spacing Changes spacing between words #em
letter-spacing Changes spacing between letters #em
line-height Changes spacing above letters, but below line above unitless ratio of font size or #px
text-align Aligns text according to parent Left, Right, Center
text-decoration used to give underline/overline/strikethrough none, underline, overline, line-through, blink
