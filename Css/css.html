CSS

CSS is the styling that changes how your webpages appear. Before, every element had to have its styling applied separately but now it can be grouped and have defaults applied for the various groupings. This allows for greater consistency, complexity, and separation of concerns

An em is a measurement of size that is equal to the size of the base font of the browser or its parent. (default 16px)
A rem is similar, but is based off of the < html> root element

Media Query: @media only screen and (feature).
Sets a specific styling for screens with a specific feature eg(max-width : 480px)
